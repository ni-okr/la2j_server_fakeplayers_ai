## Доступ и Использование

### Клонирование
```bash
git clone https://github.com/ni-okr/la2j_server_fakeplayers_ai.git
cd la2j_server_fakeplayers_ai
```

### Сборка
```bash
mvn clean compile
mvn test
mvn package
```

### Запуск
```bash
java -jar target/l2j-server-1.0-SNAPSHOT.jar
```
